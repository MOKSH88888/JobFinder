Environment Variable Diagnostic

This file helps diagnose Vercel environment variable issues.

Note: This won't show REACT_APP variables (they're built into the JS bundle).

Check the browser console on your deployed site to see the actual API URL being used.

Instructions:

  1. Open browser console (F12)
  2. Type: localStorage.clear()
  3. Go to your app homepage
  4. Try to register/login
  5. Check Network tab to see what URL it's calling
